개발목표

전국의 다양한 캠핑장을 소개하고, 실시간으로 예약할 수 있는 사이트

GitHub

https://github.com/Concat0681/CampingTogether_final

구현기능설명


-상단 검색 바 DateRangePicker를 사용 하여 입실날짜와 퇴실날짜를 입력 받고 지역과 인원을 선택해 검색할 수 있습니다.

-페이지 스크롤 시 배경화면 사진을 변경해 화면 전체 가운데를 비워 둔 후 배경화면이 커튼처럼 스크롤 내릴 시 보이게 구현했습니다.

-화면 하단에는 캠핑장 들을 신규 순, 평점 순으로 정렬 해 리스트를 구현했습니다.

구현기능설명


-상세 조건 검색 바
날짜, 인원, 캠핑장 조건을 선택 한 뒤 검색하면 Ajax를 이용해 리스트를 받아왔습니다.

-평점 순, 가격 순
상세 조건과 마찬가지로 정렬된 리스트를 Ajax로 받아왔습니다.

-지도로 보기
캠핑장 주소를 Naver geocode를 사용해 위도, 경도를 구한 뒤 Naver Map API를 사용, 캠핑장 위치를 마커에 추가해 선택하면 상세보기 페이지로 이동할 수 있게 구현했습니다.

구현기능설명


-캠핑장 좋아요
캠핑장을 유저가 자신의 리스트에 저장할 수 있습니다.

-문의하기
쪽지 함을 WebSocket을 이용해 만들어 문의하기를 보내면 받는 사용자가 바로 확인 할 수 있게 구현했습니다.

-제공서비스
캠핑장이 제공하는 서비스들을 배열에 저장해 놓은 값과 비교해 맞는 값이 있으면 아이콘으로 보여지게 구현했습니다.

-날씨정보
Open Weather Map API를 이용해 5일치 날씨정보를 캠핑장에 위치에 기반해 데이터를 받아와 날씨 정보를 구현했습니다.

구현기능설명


-전체 구성
캠핑용품을 3분류로 나누어 Slick슬라이드를 이용해 화살표를 클릭 시 캠핑용품 -> 차박 용품 -> 기타 용품 순으로 화면 전체 슬라이드를 구현했습니다.

-최신 순, 가격 순, 평점 순
캠핑용품을 정렬하는 버튼으로 Ajax로 정렬된 리스트를 받아 왔습니다.

-물품 리스트
물품 한 개를 hover하면 숨겨진 요소가 나타납니다. 그 요소를 클릭 시 숨겨놓은 Modal 창이 나타나 물품의 기본적인 정보를 확인할 수 있습니다.

구현기능설명


-상품 이미지
Slick 슬라이드 2개를 사용해 하단에 있는 슬라이드는 여러 사진을 한번에 보여주고 클릭 시 상단에 사진도 바뀌게 구현했습니다.

-수량 변경
물품의 수량 변경 시 배송비를 포함한 최종금액을 JavaScript 에서 연산한 뒤 Jsp에 그 값을 보내주었습니다.

-장바구니
장바구니 클릭 시 상품이 장바구니에 추가되고 이미 추가된 상품은 “추가되었습니다.” 버튼으로 교체하였습니다.

-상품 후기
상품의 등록, 수정, 삭제를 ajax로 처리해 최소한의 페이지 이동으로 댓글을 볼 수 있고 수정할 수 있게 하였습니다. 상품의 평점 또한 후기의 별 점이 바로 반영되게 하였습니다.

구현기능설명


-스프립을 등록하는 페이지로 사진을 여러 개 등록할 수 있고 미리보기도 구현했습니다.

-여러 개의 이미지를 JavaScript에서 업로드하기 위해 FileItem 을 사용해 업로드 하였습니다.

-주소 입력 또한 다음(daum) 주소 API를 사용하였습니다.
-내용 입력 칸은 SummerNote를 사용하여 값을 받아오고 사진 또한 업로드 할 수 있습니다.

구현기능설명


-호스트가 등록한 스프립들을 리스트 해 놓은 페이지로Carousel bootstrap을 사용하여 각 요소들마다 carousel을 적용하였습니다.

구현기능설명


-수정, 삭제 기능이 있습니다.

-자신의 강의에 날짜와 인원을 설정해 줄 수 있습니다.

-DB에 테이블을 따로 만들어 날짜와 인원의 데이터를 관리합니다.

-이를 통해 다양한 날짜의 참여인원과 설정해 놓은 인원을 검색 할 수 있습니다.
